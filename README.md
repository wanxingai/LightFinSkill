## 技能详细介绍

---

### 1. 前沿技术全景图（front-tech-map）

**定位**：全自动生成指定前沿技术领域的 SVG 全景图谱，适用于量子计算、核聚变、AI Agent 等以技术为核心的赛道。

**核心能力**：
- 🧠 领域自适应 — LLM 运行时动态分析领域特征，生成定制化搜索策略
- 🔄 双向挖掘 — 正向追踪学术源头 + 逆向溯源独角兽创始团队
- 🔗 Wikidata 师承查询 — 整合38万条博士导师关系数据
- ✂️ 强力剪枝 — 三级连接强度体系，只保留 Tier 1/2 高价值连接
- 📊 VC 分析框架 — 创始人评估 + 技术路线 + 中外对比

**输入示例**：
```
"中性原子量子计算"  /  "LLM Agent框架"  /  "核聚变磁约束"
```

**输出示例**：
```
[Topic]_TechMap_YYYYMMDD/
├── [Topic]_TechMap.svg          ← Graphviz矢量图谱
├── [Topic]_TechMap_Report.md   ← 伴随文字报告
├── generate_techmap.py          ← 生成脚本
└── nodes.json / edges.json      ← 中间数据
```

**输出样例**（部分图谱报告内容）：
> **核心发现**：中性原子量子计算赛道以 Harvard/MIT 系（QuEra）为代表，国内以中科大/清华系团队为主。溯源完成率 83%，孤儿率 12%，属于技术早期但商业化路径清晰的赛道。

### 2. 周期股研究（cyclical-stock-research）

**定位**：周期行业个股的买方研究报告，从周期象限、基本面、成本曲线、供需矩阵、预期差五个维度输出机构级分析。

**核心能力**：
- 周期象限锁定 — 去库末期 / 补库初期 / 产能出清 / 供需紧平衡 / 产能过剩
- P×Q深度拆解 — 价格/成本/毛利率十年分位点分析
- 成本曲线定位 — 全球/全国成本曲线位置 + 非对称优势量化
- 逆周期能力评估 — 管理层历史底部/顶部资本开支记录
- 预期差挖掘 — 市场共识误区 + 逆共识证据（≥3条）

**输入示例**：
```
目标公司：万华化学（600309）
关注重点：涨价逻辑、产能扩张
```

**输出示例**：
```markdown
# 万华化学 周期股研究报告

## 一、投研核心摘要

**当前周期阶段**：供需紧平衡（补库初期）

**核心逻辑亮点**：
驱动本次涨价的核心变量：MDI 全球供给收缩12%，而下游基建/家居需求增长8%，供需缺口持续扩大至2026年Q3。

**配置级别**：强力推荐

## 二、P×Q深度拆解

| 产品  | 当前价格  | 十年分位 | 单位成本 | 毛利十年分位 |
|------|----------|---------|---------|------------|
| MDI  | 18500元/吨 | 72%    | 12800元/吨 | 81%       |
| TDI  | 14200元/吨 | 65%    | 10800元/吨 | 74%       |

## 三、成本曲线定位

公司在全球MDI成本曲线中处于左侧20%分位，显著领先行业平均，现金成本领先行业约1800元/吨。
```

### 3. 可比公司分析（comparable-company-analysis）

**定位**：对目标上市公司进行同业对标分析，从业务剖析、可比公司筛选、对比分析、财务数据对比四维度输出结构化报告。

**核心能力**：
- 5-10家A股可比公司筛选，按业务相关性排序
- 竞争格局分析 — 业务对标 + 竞争焦点
- 财务数据对比 — 营收/净利/毛利率/ROE/PE/PB 全覆盖
- 综合结论 — 行业地位/核心优势/估值判断/投资启示

**输入示例**：
```
目标公司：中际旭创（300308）
分析维度：全面分析（默认）
```

**输出示例**：
```markdown
# 中际旭创 可比公司分析报告

## 二、可比公司筛选

| 分类   | 公司名称 | 公司代码 | 可比业务 | 相关业务进展 |
```

### 4. 宏观事件分析（macro-event-analysis）

**定位**：对重大宏观事件（关税/央行/地缘/汇率/大宗商品）进行六步结构化深度分析，输出机构级投研框架。

**核心能力**：
- 传导路径图 — 从事件冲击到资产定价的因果链（含时滞与放大/衰减机制）
- 历史镜像 — 2-4个可比历史案例，含量化资产回报数据
- 三情景推演 — 乐观/基准/悲观，含概率权重 + 触发条件
- 交易表达 — 具体标的/方向/仓位/入场/止损/持有周期
- 失效条件 — 3-5个可量化框架失效信号 + 应对措施

**输入示例**：
```
宏观事件：中美关税升至145%
关注资产：A股、人民币、黄金、美债
```

**输出示例**：
```markdown
# 中美关税升至145% 宏观分析

## 模块2：传导路径

关税升级 → 出口企业盈利预期下调（1-4周）
                    ↓
              人民币贬值压力（1-3天） → 外资流出（1-4周） → A股整体承压
                    ↓
              避险情绪升温（即时） → 黄金上涨（即时） → 美债收益率下行（1-3天）

## 模块4：情景推演

| 情景 | 概率 | 触发条件 | A股方向         | 黄金方向  | 美债方向      |
|------|------|---------|----------------|----------|-------------|
| 乐观  | 20%  | 谈判重启，关税回落至60% | 沪深300上涨5-8% | 回落至2200 | 收益率下行至3.8% |
| 基准  | 55%  | 维持当前关税水平       | 沪深300下跌5-10% | 维持2500  | 维持4.1%     |
| 悲观  | 25%  | 关税全面扩大至200%+  | 沪深300下跌15%+ | 突破3000  | 收益率突破5%  |

## 模块5：交易表达

| 编号 | 标的               | 方向 | 仓位 | 入场条件            | 止损        | 持有周期 |
|------|------------------|------|------|-------------------|------------|---------|
| T1   | 沪深300ETF(510300) | 空   | 40%  | 沪深300跌破3600点   | 沪深300回升至3850 | 4-8周   |
| T2   | 黄金ETF(518880)     | 多   | 30%  | 黄金回调至2300美元/盎司 | 黄金跌破2150  | 1-3月   |
```

### 5. 事件点评（event-commentary）

**定位**：突发市场事件的买方视角深度点评，帮助投资者快速识别预期差、挖掘Alpha机会。

**核心能力**：
- 预期差识别 — 市场共识阶段分析 + 逆共识证据
- 定量敏感度分析 — 核心变量±10%波动对EPS的影响
- Alpha机会矩阵 — 纵向供应链穿透 + 业务纯度对比
- 隐性受益标的 — 替代品/配套耗材/期权价值黑马挖掘
- 蝴蝶效应 — 跨界意外触发行业价值重估

**输入示例**：
```
事件：美国对华加征60%关税
时间：2026-04-18
```

**输出示例**：
```markdown
# 美国对华加征60%关税 事件点评

## 三、Alpha机会全景矩阵

### 纵向供应链穿透

上游资源 → 中游制造 → 下游应用
  铝材       铝加工      汽车零部件
  铜材       覆铜板      PCB厂商
                    ↓
              价值量转移：从中游制造向下游国产替代转移

### 核心配置清单

| 环节/标的        | 代码     | 逻辑厚度 | 目标位/关键验证点           |
|----------------|---------|--------|------------------------|
| 首选弹性标的     | 公司A    | 一阶受益：国产替代加速，订单转移 | Q2财报营收增厚30%验证   |
| 预期差标的       | 公司B    | 市场误判为受损方，实则转单受益 | 海外客户认证公告催化剂   |
| 隐性受益标的      | 公司C    | 替代需求爆发，产能利用率从60%→85% | 产能利用率数据验证    |
```
